# Doctor Uncle AI (Frontend)

Doctor Uncle is an intelligent, AI-driven digital health assistant application. This frontend connects users with an easy-to-use symptom checker, dynamically asking medical questions and concluding with a highly detailed diagnosis/suggestion schema generated by OpenAI on the backend.

## Tech Stack
- **Framework**: [React 18](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **UI & Styling**: [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)
- **Data Fetching & Routing**: React Query, React Router DOM

## Running the Application

### 1. Configure Environments
Because this is a Vite application, environment variables must be prefixed with `VITE_`.
Create a `.env` file at the root of the project with your backend API pointer:
```env
VITE_API_BASE_URL=http://localhost:5000/api/v1
```

### 2. Install & Start
```sh
# Install dependencies
npm install

# Start the Vite development server
npm run dev
```

## Features
- **Interactive Symptom Input**: Users can select quick options or describe their symptoms using their own words.
- **Dynamic Question Flow**: Connects to the backend AI to prompt appropriate diagnostic follow-ups dynamically.
- **Structured Health Insights**: Presents comprehensive analytics—including potential conditions, home remedies, triage severity, and specialist recommendations—via detailed and visually distinct UI cards.
